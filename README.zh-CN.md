## 这是什么

你被丢进一片开阔的战场，手里只有一把手枪，面前是越涌越多的丧尸。枪会自动朝鼠标方向开火，所以整局游戏都靠你的走位：拉扯尸群、撕开缺口、捡起击杀掉落的金币和经验。每升几级就三选一拿一个强化，而你一路拼出来的 Build，决定了你能不能活到 Boss 面前。

开局像在打扫战场，到后面会变成满屏几百只丧尸、还有一个会召唤援军的 Boss。

## 为什么好玩

- **自动开火，全靠走位。** 不用换弹、不用点射，注意力全花在站位和拉扯上。上手简单，玩好很难。
- **战术 HUD 更容易读。** 阶段、时间、威胁、金币、血量、主武器和物品栏被放进清晰的战斗优先级区域，不再互相抢注意力。
- **开局一分钟更友好。** 前期刷怪压力会渐进上升，新手拾取范围更大，进入后续阶段时还能拿到明确的金币和回血奖励。
- **装备不再是 Emoji。** 商店消耗品和增益现在使用统一的 96×96 透明战术道具图标，底部物品栏也同步显示真实图片。
- **动作和深度更强。** 主角更大，角色按脚底排序，武器和持有装备会作为手部独立层渲染，主角开火有后坐力，拾取拖尾和环境色调让战斗更有实体感。
- **五个递进阶段。** 难度是一档一档往上抬的，从零星几只行尸到密不透风的尸潮，每进一个阶段都能明显感觉到压力收紧。
- **有完整技能组的 Boss。** 母巢暴君会预警登场、打出环形弹幕、震地放冲击波、还会召唤援军。这是一场真正的战斗，不是个血厚的木桩。
- **构筑多样。** 六把武器（各有 6 级进化）、一整套被动属性、商店消耗品，外加本局限定的主动技能——每一局都能凑出不一样的组合。
- **真正由你操控的主动技能。** 疾冲穿过尸群、爆破清场、开屏障扛伤、放迟滞拖慢时间。中途在商店购买，绑定 `Z` `X` `C` `V`，HUD 上实时显示冷却。
- **打起来很爽。** 屏幕震动、命中闪光、枪口火花、冲击波、角色跑动动画、残留的尸体和血迹——全部跑在朴素的 Canvas 2D 上，满屏怪也依然流畅。

## 敌人

| | |
|---|---|
| **行尸** | 最基础的步行者。慢，但不停，而且永远有更多。 |
| **疾跑者** | 拉近距离极快，专治不好好留距离的玩家。 |

| **自爆体** | 冲过来就引爆，得趁早打掉，否则吃亏。 |
| **壮汉** | 一堵血墙，几乎不吃击退。 |
| **母巢暴君** | Boss。弹幕、震地、召唤——撑过它就赢了。 |

## 操作

| 行为 | 输入 |
|---|---|
| 移动 | `W` `A` `S` `D` / 方向键 |
| 瞄准 | 鼠标（自动开火） |
| 使用道具栏 | `Q` `E` `R` |
| 主动技能 | `Z` 疾冲 · `X` 冲击爆破 · `C` 能量屏障 · `V` 时间迟滞 |
| 打开商店 | `B` |
| 选择升级 | `1` / `2` / `3` |
| 开始 / 重开 | `Space` |

## 商店

击杀掉落金币，随时按 `B` 花掉。前期商店卖消耗品——手雷、治疗、护盾、临时增益。从第 3 阶段起解锁主动技能卡：买下后本局永久持有，冷却会显示在 HUD 上。

## 本地运行

```bash
npm install
npm run dev
```

打开 Vite 输出的本地地址（默认 `http://localhost:5173`）。

| 命令 | 作用 |
|---|---|
| `npm run dev` | 启动带热更新的开发服务器。 |
| `npm run build` | 先类型检查，再构建到 `dist/`。 |
| `npm run preview` | 本地预览生产构建。 |
| `npm test` | 运行测试（单元测试 + 无头游戏模拟）。 |

## 技术栈

使用 TypeScript + Vite，渲染基于 Canvas 2D，核心是一套自研的轻量 ECS。整套模拟是确定性的——种子随机数加上实体/组件存储——因此无头测试可以脱离浏览器完整复现一整局。游戏数据（敌人、武器、技能）在加载时用 Zod schema 校验。

```
src/
├── ecs/        实体/组件存储 + 种子确定性随机数
├── systems/    移动、生成、战斗、武器、拾取、装备、技能
├── render/     Canvas 渲染、资源加载、精灵尺寸
├── data/       数值、敌人、武器、被动、装备、技能
├── fx/         粒子、尸体、血迹
├── sim/        无头模拟器（复用真实系统）
├── ui/         DOM 覆盖层（标题、HUD、升级、商店、结算）
└── game.ts     状态机、系统流水线、世界渲染
```
